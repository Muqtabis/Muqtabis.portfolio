# Muqtabis Ahmar - Portfolio

A personal portfolio website showcasing my projects, skills, and experience as a Full-Stack Developer.

## Tech Stack

- React
- TypeScript
- Tailwind CSS
- Vite

## Features

- Responsive design
- Dark/Light theme toggle
- Downloadable resume
- Project showcase
- Contact information

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
